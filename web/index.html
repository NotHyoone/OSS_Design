분석 시작

GitHub ID를 입력하면 역량 분석을 시작합니다.

github.com /

영문, 숫자, 하이픈(-)만 사용 가능합니다.

분석에는 일반적으로 10 ~ 60초가 소요됩니다.

활동성

커밋 빈도, 저장소 업데이트 빈도를 기반으로 얼마나 꾸준히 활동하는지 측정합니다.

🧩

기술 다양성

사용 언어 수와 프로젝트 유형 다양도를 분석해 기술 스택 폭을 평가합니다.

🤝

협업도

PR · Issue · Fork 참여율을 측정해 오픈소스 및 팀 협업 역량을 파악합니다.

🔄

지속성

활동 기간, 갭 분석, 연속성을 종합해 개발자로서의 꾸준함을 평가합니다.
