GitHub으로 로그인

GitHub OAuth 2.0으로 인증하면 분석 이력이 계정에 저장되어 언제든지 재조회할 수 있습니다.

OAuth 미설정 안내
서버에 GitHub OAuth App이 설정되지 않았습니다.
로그인 없이도 GitHub ID를 직접 입력해 분석을 진행할 수 있습니다.
GitHub으로 로그인

로그인하면 이런 기능을 사용할 수 있습니다:

  • 분석 이력 영구 저장 및 조회
  • 시계열 성장 추이 비교
  • 내 GitHub 계정 자동 연동 분석
로그인 없이 계속하려면 홈으로 이동하세요.
