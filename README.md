# OSS_Design

오픈소스SW설계 프로젝트

## 프로젝트 소개

이 저장소는 오픈소스SW설계 과목의 팀 프로젝트를 위한 저장소입니다.

## 디렉토리 구조

```
OSS_Design/
├── src/            # 소스 코드
├── tests/          # 테스트 코드
├── docs/           # 문서
├── CONTRIBUTING.md # 기여 가이드
├── LICENSE         # 라이선스
└── README.md       # 프로젝트 설명
```

## 시작하기

### 필요 환경

- Python 3.8 이상

### 설치 방법

```bash
# 저장소 클론
git clone https://github.com/NotHyoone/OSS_Design.git
cd OSS_Design

# 의존성 설치
pip install -r requirements.txt
```

### 실행 방법

```bash
python src/main.py
```

### 테스트 실행

```bash
python -m pytest tests/
```
