# EngReadMate App
영어로 학습하는 모든 직종을 위한 전문 문서 읽기 도우미 앱

EngReadMate는 공학, 의학, 경영, 법학 등 다양한 분야의 영어 전문 문서를 더 쉽게 이해할 수 있도록 돕는 앱 기반 학습 지원 서비스입니다.
사용자는 영어 문장이나 문단을 입력한 뒤 직무 맥락 기반 단어 설명, 문장 번역, 문장 구조 분석, 문단 요약 기능을 통해 원문을 효율적으로 학습할 수 있습니다.
또한 학습한 단어를 개인 단어장에 저장해 반복 복습할 수 있으며, 영어 원문 자료를 자주 읽는 대학(원)생, 연구자, 실무자, 직무 전환 준비 학습자를 주요 대상으로 합니다.

## App Version Notes

- 본 저장소는 앱 버전 MVP 기준으로 구성되어 있습니다.
- UI는 앱형 화면 구성(입력/결과/단어장)으로 설계되어 있습니다.
- 기능은 로컬 환경에서 즉시 실행 가능한 형태로 구현되어 있습니다.
