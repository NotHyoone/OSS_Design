## 프로젝트 개요

GitHub Activity Insight는 개발자의 GitHub 활동 데이터를 분석하여 역량을 해석하고, 개인 맞춤형 피드백과 성장 방향을 제시하는 웹 기반 시스템입니다.

기존의 단순한 저장소 수나 커밋 수 중심 평가를 넘어서, 언어 사용 비율, 프로젝트 경험, 협업 지표, 활동 패턴 등을 종합적으로 분석해 보다 의미 있는 인사이트를 제공하는 것을 목표로 합니다.

## 프로젝트 유형

- 웹 기반 분석 서비스
- GitHub 데이터 연동형 대시보드 및 리포트 시스템

## 핵심 기능

- GitHub 계정 및 활동 데이터 수집
- 언어 사용 비율 및 프로젝트 규모 분석
- 커밋 패턴 기반 활동성 분석
- 협업 지표 및 기술 스택 프로파일 평가
- 개발자 유형 분류 및 개인 맞춤형 피드백 제공
- 시각화 대시보드 및 PDF 리포트 생성

## 대상 사용자

- 취업을 준비하는 컴퓨터공학 전공 학생
- 자신의 개발 역량을 점검하고 싶은 개발자
- GitHub 포트폴리오를 체계적으로 개선하고 싶은 사용자

## 기대 효과

- GitHub 활동을 기반으로 개인 역량을 객관적으로 파악
- 정량 데이터에 해석을 더한 실질적 피드백 제공
- 취업 및 포트폴리오 개선에 활용 가능한 분석 리포트 제공
